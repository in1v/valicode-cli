# Changelog

## 2.0.0

- Initial public PyPI release of the Valicode CLI.
- Adds `valicode login`.
- Adds workspace scanning with `valicode scan`.
- Adds staged and diff analysis with `valicode analyze`.
- Defaults to the production API at `https://api.valicode.sbs/v1`.
- Supports API base override through `VALICODE_API_BASE` or `valicode config set api_base`.
